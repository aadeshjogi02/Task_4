Project

Virtual Human Computer Interaction

The challenge is to develop a hands-free Human-Computer Interaction (HCI) system that enables amputees and individuals with limited hand mobility to operate computers effectively. The system needs to accurately interpret eye movements and convert them into meaningful cursor.The "Virtual Human-Computer Interaction" project introduces an innovative system combining facial landmark detection, mouse control, and virtual keyboard functionality.Leveraging OpenCV, MediaPipe, Computer Vision, and PyAutoGUI, this project creates a dynamic interface that enables users to interact with their computers using facial gestures.Facial landmark detection, powered by the MediaPipe Face Mesh, accurately tracks key points on the user's face, allowing precise mouse cursor control.
